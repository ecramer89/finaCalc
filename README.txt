---Financial Calculator---

Before attempting to run app or unit tests, please install required dependencies:

cd into project root directory 

run command:

npm install


--RUN UNIT TESTS--

cd into project root directory

run command

npm run test
(one should be skipped; this is expected. Please read comments above test for explanation)

--RUN APPLICATION--

cd into project root directory
run command
npm run start

the application is running on: localhost:3000
